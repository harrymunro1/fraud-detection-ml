# 💳 Fraud Detection using Machine Learning

## 📌 Overview
This project builds a fraud detection model using machine learning techniques on an imbalanced dataset. The focus is on robust evaluation, handling class imbalance, and optimising model performance for real-world use.

---

## 🧠 Key Features
- Handling imbalanced data using SMOTE
- Model comparison:
  - Logistic Regression
  - Random Forest
  - XGBoost
- Cross-validation for robust evaluation
- Hyperparameter tuning
- Threshold optimisation for business use

---

## 📊 Results
- Models converged to ~0.75 PR-AUC using cross-validation
- Initial higher scores were attributed to variance in train-test split
- Threshold optimisation enabled control over precision vs recall

---

## 💼 Business Impact
- Demonstrates real-world fraud detection trade-offs
- Highlights importance of evaluation methods
- Shows how models can be tuned to business objectives

---
